# 📊 Sales Dashboard | Power BI

## 📌 Project Overview
This Power BI Sales Dashboard provides a complete analysis of sales performance across different years, product categories, and customer locations.  
The dashboard helps identify revenue trends, profit contribution, top-selling products, and customer demand patterns.

It supports data-driven decision-making for:
- revenue optimization
- product strategy
- customer analysis
- yearly growth tracking
- regional sales performance

---

## 🛠 Tools & Technologies
- Power BI
- Excel / CSV Dataset
- DAX Measures
- Data Cleaning
- Data Modeling
- Business Intelligence

---

## 📂 Dataset Used
The dashboard uses sales-related business tables including:
- Customers
- Orders
- Order Details
- Products
- Product Lines
- Payments
- Employees
- Offices

---

## 🧩 Data Model
A relational data model was created to connect customers, orders, products, and payments tables for accurate KPI calculations and business insights.
